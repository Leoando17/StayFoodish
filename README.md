# StayFoodish

StayFoodish is a static prototype for an AI-powered personal nutrition and zero-waste food management app.

## What is included

- Pantry inventory tracking
- Expiring-soon food alerts
- Recipe suggestions based on available ingredients
- Three-day meal plan preview
- Smart shopping list gaps
- Local demo data stored in the browser

## How to run

Open `index.html` in a browser. No build step or backend is required.
